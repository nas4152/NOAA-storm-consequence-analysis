dmgdata <- select(stormdata, year, STATE, EVTYPE, INJURIES, FATALITIES,
                   PROPDMG, CROPDMG)
tstmwind <- unique(grep("TSTM", dmgdata$EVTYPE, value = TRUE))
tstmwind <- tstmwind[tstmwind != 'NON-TSTM WIND' & tstmwind != 'NON TSTM WIND'
                     & tstmwind != "TORNADOES, TSTM WIND, HAIL"]
dmgdata$EVTYPE[dmgdata$EVTYPE %in% tstmwind] <- 'THUNDERSTORM WIND'
dmgdata$EVTYPE[dmgdata$EVTYPE == 'HEAT'] <- 'EXCESSIVE HEAT'

In order to determine the most damageing storm types in regard to public health and property damage, data for injuries, fatalities, property and crop damage were selected. Data was analyzed looking at the whole year to limit effects of seasonal change and determine overall impact so only the year of the event was kept for analysis. This analysis focuses on the impact across the US, however state designation was kept in the rearranged dataset as it may be useful in further analysis for more focused policy decisions.

ymedian <- median(dmgdata$year)
dmgdata <- filter(dmgdata, year >= ymedian)

The full dataset includes entries from the years 1950 - 2011. However, the median year is 2002. This indicates that half of the data falls within the last decade of six. The analysis focuses on the data collected from the median year and later to limit the effect of increased record collection over time.

Results

Ranking Storms by Damage Parameters

library(reshape2)
dmgmelt <- melt(dmgdata, id.vars = c("year","STATE", "EVTYPE"))
dmgsum <- dcast(dmgmelt, year + EVTYPE ~ variable, sum)

First yearly sums were calculated for each storm type.

meandmg <- summarise(group_by(dmgsum, EVTYPE), injury = mean(INJURIES), 
                   fatality = mean(FATALITIES), prop = mean(PROPDMG), 
                   crop = mean(CROPDMG))
topinj <- arrange(meandmg, desc(injury))[1:5,1]
topftl <- arrange(meandmg, desc(fatality))[1:5,1]
topprop <- arrange(meandmg, desc(prop))[1:5,1]
topcrop <- arrange(meandmg, desc(crop))[1:5,1]

In order to determine the most damaging storm types, the yearly sums were averaged across the data for each parameter. The storm types with the highest mean for each value were stored.
